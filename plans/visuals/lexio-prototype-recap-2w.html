07 — Mental model

Mental model essentials

Four things to internalize before opening the codebase.

Core is pure TS

core/ never imports React, Next, or Dexie. SM-2 lives there. Test it without a browser.

Stub auth, banner mandatory

Stub auth lives in localStorage via Zustand persist. NOT-PROD banner is mandatory on every authenticated route.

Ports and adapters

Repository ports in core/ports/. Dexie impls in lib/storage/. Mock API in lib/api/ wraps the same repos with simulated latency — swap to HTTP when backend lands.

Brand §12 is non-negotiable

Zinc + Indigo only, dark-first, no gradients, no glassmorphism, ≤200ms animations, sentence case.

08 — Hotspots

Cognitive debt hotspots

Known rough edges to address before v0.2 ships.

E2E flakiness

Playwright happy-path races on Zustand auth-rehydration in CI. Marked continue-on-error: true. Stabilize with real auth in v0.2.

Lighthouse audits public routes only

/dashboard requires auth wall. Inject storageState in CI when backend lands.

18 ESLint boundary suppressions

All for intra-feature barrel re-exports. Acceptable; revisit config in v0.2.

Dexie v1 → v2 migration single-shot

Card entity bumped during phase-13 for §4.2 fields. Verify migration on real user data when there is any.
